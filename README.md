# TravelTrucks 🚐

A web application for renting camper vans, built with Next.js and TypeScript.

## 🔗 Links

- [Live Demo](https://travel-trucks-app-yltv.vercel.app/)
- [Repository](https://github.com/MarinaYastrebova/travel-trucks-app)

## 📋 Description

TravelTrucks is a camper van rental service that allows users to browse available campers, filter by
location, vehicle type, engine and transmission, view detailed information and book a camper.

## ✨ Features

- Browse catalog of available camper vans
- Filter campers by location, body type, engine and transmission
- Load more campers with pagination
- View detailed information about each camper
- Image gallery with Swiper thumbnails
- User reviews with star ratings
- Booking form with validation and notifications

## 🛠️ Tech Stack

- [Next.js 15](https://nextjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [TanStack Query](https://tanstack.com/query)
- [Swiper](https://swiperjs.com/)
- [React Icons](https://react-icons.github.io/react-icons/)

## 🚀 Getting Started

### Installation

1. Clone the repository:

```bash
git clone https://github.com/MarinaYastrebova/travel-trucks-app.git
cd travel-trucks-app
```

2. Install dependencies:

```bash
npm install
```

3. Run the development server:

```bash
npm run dev
```

4. Open (http://localhost:3000) in your browser.
